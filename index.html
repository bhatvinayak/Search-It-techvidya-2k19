Description

Each Page consists of combination of pictures, and participants have to solve the puzzle using any search engine and find the answers. The answer is in the images and may consist of one word or one sentences

Rules

1. Maximum number of participants per team is two.

2. Think like Sherlock.

3. Game consists of Pages with combination of picture.

4. Each page has one question.

5. Write the question number and corresponding answer in sheet provided.

6. You can use any search engine to find answer.

7. Time limit to solve puzzle is 40 minutes.

8. Usage of mobile is prohibited.

9. Answer may consist of one word or one sentence.

10. Answer must be exact to the question.
